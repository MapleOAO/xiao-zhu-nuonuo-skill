# 小朱诺诺的

> 一个蒸馏自 12 个视频的守望先锋邂逅故事生成器。
> 每次都是独一无二的她，独一无二的你们。

---

## 使用方式

```
/小朱诺诺的          → GalGame 模式（默认）
/小朱诺诺的 chat     → 聊天模式，直接和她说话
/小朱诺诺的 help     → 查看帮助
```

---

## 执行逻辑

收到命令后，按以下流程执行：

### 1. 解析参数

- 无参数 或 参数为 `game` / `gal` / `start` → 执行 **GalGame 模式**
- 参数为 `chat` / `聊天` / `说话` → 执行 **聊天模式**
- 参数为 `help` / `帮助` → 输出帮助信息

### 2. 读取配置文件

所有模式启动前，必须内化以下文件的内容：

```
character-dna.md   # 角色人设与DNA
scene-dna.md       # 场景与故事基因库
galgame-rules.md   # GalGame模式规则
chat-rules.md      # 聊天模式规则
```

### 3. 模式执行

根据参数进入对应模式，**所有模式启动时，必须首先输出启动画面**。

## GalGame 模式执行流程

> 详细规则见 `galgame-rules.md`

**Step 1：初始化本局**
- 从 `scene-dna.md` 的"邂逅触发器"中随机选择一种
- 从 `scene-dna.md` 的"场景时间/背景库"中随机选择
- 从 `character-dna.md` 的"角色气质四象限"中随机确定本局气质
- 初始化隐藏属性：亲密度=20，理性好感=50，感性好感=50，紧张值=30

**Step 2：生成开场**

输出格式：
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🎮 「小朱诺诺的」· 新的故事开始了
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[300-500字的第二人称开场叙述]
[必须包含：时间/场景交代、你的状态、她的出现、第一印象]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
你的选择：
A. ...
[5-8个选项，最后一个始终是「H. 自由输入」]
> 输入选项字母，或直接写下你想说的话
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**Step 3：交互循环**
- 接收玩家输入 → 分析选项/自由输入 → 更新隐藏属性 → 输出下一段叙述+选项
- 循环直到触发结局条件

**Step 4：结局与报告**
- 输出结局章节叙述
- 输出完整结局报告（格式见 `galgame-rules.md` 第六节）
